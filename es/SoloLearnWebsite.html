Párrafos

La función Párrafo permite escribir texto y darle formato de párrafo. En el párrafo, puede utilizar diferentes etiquetas para escribir sus palabras, como por ejemplo, ponerlas en negrita o en cursiva. Incluso puede cambiar el color de las palabras para que diga rojo o azul usando la etiqueta de fuente <font></font> y el atributo de color. Otra que puede hacer es proporcionar enlaces a otras páginas web o sitios web como este enlace. Lo último de lo que hablaré para los párrafos es la alineación de los mismos. Mediante el atributo align puede alinear los párrafos a la izquierda, al centro o a la derecha.

Imágenes

Las imágenes se pueden insertar utilizando la etiqueta de imagen <img> y el atributo src. Se puede realizar una modificación adicional agregando atributos como altura, anchura y alt. La etiqueta alt sólo es necesaria para aquellos que no pueden ver la imagen, como las personas que tienen malas conexiones a Internet o utilizan un texto para escupir el dispositivo.

Listas

Listas Ordenadas

  1. Las listas ordenadas usan números para mostrar diferentes elementos
  2. La etiqueta es <ol></ol>
  3. Para agregar elementos, debe usar la etiqueta <li></li>
  4. Usa esto para cuando publiques pasos para algo o publiques artículos de mayor a menor o de menor a mayor.

Listas Desordenadas

  • Las listas desordenadas usan viñetas para enumerar elementos.
  • La etiqueta es <ul></ul>
  • Agregar elementos es lo mismo que las listas ordenadas
  • Use esto para cuando solo desee enumerar los elementos y no le importe el orden en que entran.

Listas Descriptidas

Etiquetas
Utiliza la etiqueta <dl></dl> para iniciar la lista.
Artículos
Utiliza <dt></dt> para agregar términos para elementos y <dd></dd> para agregar detalles al término.

Listas Anidadas

  1. Las listas anidadas son listas con otras listas en ellas.
    • Puede ser con una lista ordenada o no ordenada.
    • Tienes que poner otra lista dentro de la <li></li> etiqueta donde quieres que vayan.
  2. Las listas anidadas no son listas que utilizan varias listas colocadas una al lado de la otra.

Mesas

Las tablas son tablas. Las tablas son simplemente texto puesto debajo o uno al lado del otro.
Las tablas se utilizan para ordenar datos como en Microsoft Excel. Las tablas pueden tener colores de fondo
